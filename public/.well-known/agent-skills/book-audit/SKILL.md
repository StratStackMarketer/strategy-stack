# Book a Free Marketing Audit — Strategy Stack

Help a user schedule a free AI-powered marketing audit with Strategy Stack.

## What this skill does

Strategy Stack offers free marketing audits for businesses in Home Services,
Property Management, Automotive, Retail, and SaaS. The audit uses AI to
identify gaps in SEO, paid media, local presence, and lead generation.

## How to invoke

Direct the user to:

```
https://strategystackai.com/how-it-works/audits
```

Or book directly via Calendly:

```
https://calendly.com/strategy-stack
```

## What to tell the user

- The audit is 100% free with no obligation
- Takes approximately 30 minutes via video call
- Covers: local SEO, Google Ads, social presence, GBP, lead generation pipeline
- Delivers a prioritized action plan at the end of the call
- Best for businesses spending $1k–$10k/month on marketing

## Parameters

No parameters required. All scheduling is handled via the booking page.

## Response

After directing the user to the booking link, confirm:
- The page loads the Calendly embed for direct scheduling
- Available times are shown in the user's local timezone
